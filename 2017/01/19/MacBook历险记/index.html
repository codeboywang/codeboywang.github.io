密码忘了

说起来十分搞笑，这台mac是爸爸给我激活的，我和他口述了密码。然而我忘了，爸爸也忘了…………
等我回来时，两个人都一脸懵逼。😓
我试着重装系统，结果莫名其妙的不动了，一个下午进度条没动，显示还剩6分钟，这是6分钟还是6天啊。
于是我注册了新的Apple ID，注销了原账户…………这过程中有很多神奇的操作，我也记不得太清楚。、

Touch ID挂了

好不容易可以正常使用mac了，我抱着万分激动的心情去录入指纹，结果出现了如下问题：
Cancelled by another authentication
于是进过艰苦卓绝的探索，终于搞定了
Apple论坛传送门

HEXO搭建博客

还好之前在慕课上看过Linux的课程，也算能看懂操作命令了。
磕磕碰碰遇到一些问题。总结一下命令操作顺序：

  • 安装Node.js
  • 安装Git
  • 由于Mac没带Java，于是顺便装一下
  • 新建目录，进入该目录
  • sudo npm install hexo-cli -g –save
  • hexo init
  • npm install
  • 主题下载maupassant
  • git config –global user.name “yourname”
  • git config –global user.email “youremail”
  • 设置config
    deploy:
    type: git
    repo: git@github.com:yourname/yourname.github.io.git
    branch: master
  • npm install hexo-deployer-git –save来安装所需的插件
  • GitHub设置ssh

现在唯一的问题就是多说的评论框出问题了。我把short_name给忘了……
补充，我想起来了，是二级域名……

Windows10双系统

  • 先在传送门下载镜像就可以了。
  • 在Luangchpad的“其他”中有bootcamp，安装windows
  • 在开机时同时按住option，可以选择启动的系统
  • 在“偏好设置”中的启动磁盘中可以设置启动的系统磁盘
  • 上一步设置为mac后，可以再用bootcamp抹去windows磁盘

然而我的windows出了一些问题，他没有无线网，根本检测不到。于是手机下载驱动精灵，然后再下载无线网驱动，安装好了还是检测不到。于是放弃了，先缓一缓吧……
