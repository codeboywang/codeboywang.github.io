HDU_5833

题意

给一堆数，任意选，连乘得一个平方数。问有几种解。

思路

2000以内有303个只因子，把每个数拆成质因子之积，若为偶数当做0，奇数当做1。
例如12 24 18这三个数组成矩阵是
0 0 1 (因子2)
1 1 0 (因子3)
0 0 0 (因子5)
0 0 0 (因子7)
…… 303行
高斯消元亦或求秩，答案是2^(n-秩)-1
